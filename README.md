SHORT DESCRIPTION ABOUT THE PROJECT: 
In this project, we created a responsive landing page using HTML and CSS. We built the basic HTML5 structure and linked an external CSS file for styling. A header section was designed with a logo and navigation links using Flexbox for alignment.
We then created a hero section containing a main heading, a short description, and a call-to-action button, along with a background image and overlay for better visual appeal. A footer section was added at the bottom with basic information and links.
Flexbox was used for layout and alignment, and media queries were implemented to make the webpage responsive across different screen sizes such as mobile, tablet, and desktop.
